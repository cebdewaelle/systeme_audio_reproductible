# systeme_audio_reproductible

## Execution de la pipeline
```sh
python main.py --pipeline pipeline_example.yaml
```

## Liste des operations supportés

- append: ajoute un son après un autre
- overlay: ajoute un son par dessus un autre
- fade: baisse le volume sonore de la piste audio puis le réaugmente à un timecode donné
- cut: coupe un fichier à l'endroit désigné
